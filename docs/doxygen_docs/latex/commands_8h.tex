\subsection{Detailed Description}
Definitions for Q\-B Move commands, parameters and packages. This file is included in the Q\-B Move firmware, in its libraries and applications. It contains all definitions that are necessary for the contruction of communication packages.

It includes definitions for all of the device commands, parameters and also the size of answer packages.
