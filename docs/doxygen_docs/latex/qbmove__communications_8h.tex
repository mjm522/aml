\subsection{Detailed Description}
Library of functions for S\-E\-R\-I\-A\-L P\-O\-R\-T communication with a Q\-B Move. Function Prototypes. This library contains all necessary functions for communicating with a Q\-B Move when using a U\-S\-B to R\-S485 connector that provides a Virtual C\-O\-M interface.
